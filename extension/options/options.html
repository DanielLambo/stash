Snippets

Enable snippet expansion
Type a trigger anywhere on the web and it expands to your saved text. Supports {cursor}, {date}, {date:FORMAT}, {clipboard}, and {input:Label}.
Your snippets
Triggers must be unique. A short prefix like ; or ;; avoids accidental matches.

Vault (encrypt sensitive copies)

Auto-detect & encrypt sensitive clips
Detects API keys, JWTs, credit cards, SSNs, and other secrets. Encrypts with AES-GCM using a key derived from your master password (PBKDF2-SHA256, 600 000 iterations).
Master password
Not set up. Choose a password to enable encryption.
